Basic Glyph Types

Ω
Power Glyph
DESCRIPTION
Boost all thing Antimatter related, usually stuff in the Antimatter Dimension subtab
EFFECTS
Antimatter Dimension power +X
Antimatter Dimension multiplier ×X
Dimension Boost multiplier ×X
Buy 10 bonus +X
∞
Infinity Glyph
DESCRIPTION
Boost all thing Infinity related, not very good
EFFECTS
Infinity Dimension power +X
Infinity Point gain ×X
Infinity gain ×X
Infinity Power conversion rate +X
Ξ
Replication Glyph
DESCRIPTION
Boost all thing Replicanti related, a game changer really
EFFECTS
Replicanti multiplier power +X
Replication speed ×X
Dilated Time gain per 1e10000 Replicanti +X
Replicanti glyph factor exponent +X
Δ
Time Glyph
DESCRIPTION
Boost all thing Eternity related, also bend time itself
EFFECTS
Time Dimension power +X
Eternity Point gain ×X
Eternity gain ×X
Time speed ×X
Ψ
Dilation Glyph
DESCRIPTION
Boost all thing Dilation related, not really recommended early on
EFFECTS
Antimatter Dimension power while Dilated +X
Time Theorem per hour +X
Dilated Time gain ×X
Tachyon Galaxies threshold multiplier ×X
Ϟ
Reality Glyph
DESCRIPTION
Bend Reality itself, can only be use once yet that is more than enough
EFFECTS
Glyph effective level +X
Galaxies strength +X%
Upgrade Amplifier ^X
Dilated Time glyph factor exponent +X
